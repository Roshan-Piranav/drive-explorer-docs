# Installation Guide

This guide helps you install the _Drive Explorer Pro_ extension within Google Sheets.

## 1. Prerequisites

- A Google account  
- Access to Google Sheets

## 2. Installing the Extension

To install the Drive Explorer Pro extension:

1. Open any **Google Sheet**.  
2. Click on the top menu:  
   **Extensions → Add-ons → Get Add-ons → Search for Drive Explorer Pro By Thinksolv → Install**

3. The extension will be added automatically to your Google Sheets.

**You should now see _Drive Explorer Pro_ listed under the Extensions menu.**

> Once installed, you can start browsing, managing, and exporting Drive files directly from within Google Sheets.
