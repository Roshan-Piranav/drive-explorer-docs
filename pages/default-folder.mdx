# Setting a Default Folder in Drive Explorer Pro

## Purpose of Setting a Default Folder

Setting a default folder in Drive Explorer Pro allows you to streamline repetitive tasks by predefining a specific Google Drive folder to work with. Once set, this folder acts as the default source or destination for operations such as file listing, pushing.., eliminating the need to select the folder manually each time.

This is particularly useful when working with:

- Frequently accessed project folders
- Shared team directories
- Automated reporting or file tracking

By designating a default folder, you ensure consistency, save time, and reduce the chance of selecting the wrong location.

## How to Set or Change the Default Folder

Follow the steps below to configure or update the default folder:

## Steps to Set the Default Folder

1. Open the Google Sheet where you want to use Drive Explorer Pro.  
2. Navigate to **Extensions → Drive Explorer Pro → Open Drive Explorer** to open the sidebar.  
3. In the sidebar, click the **Change** button under the default goofle drive folder section.  
4. Browse your Google Drive and select the folder you wish to set as the default.  
5. Click the **Select** Button to confirm and save your selection.

> Once selected, the folder will be saved as the default and used automatically for supported actions unless changed.

## Updating the Default Folder

You can change the default folder at any time by repeating the steps above and selecting a different folder. The new folder will replace the previous one immediately upon confirmation.

---

> Tip: Set or update the default folder when your workflow shifts to a new project or directory to maintain consistency.
