# Image Preview Settings

You can customize how image previews appear in your spreadsheet by adjusting the preview cell dimensions. This setting is available under the **Settings** tab in the Drive Explorer Pro sidebar.

## Options Available

- **Resize image preview cells**  
  Enable this option to control the size of cells where image thumbnails are displayed.

- **Cell Width (px)**  
  Set the desired width for each image preview cell.

- **Cell Height (px)**  
  Set the desired height for each image preview cell.

These settings help maintain a consistent and organized layout when displaying images in your spreadsheet.

> Once you adjust the preview sizes, click Save selected settings as Default at the bottom of the sidebar. This ensures your chosen image sizing will be applied automatically in future sessions, as shown in the sidebar layout.
